🚀 Engine Core

AI Agent Orchestration System - Pregel & Actor Model Framework

Quick Start

Get started in minutes:

pip install engine-core

from engine_core import AgentBuilder

agent = AgentBuilder() \
    .with_id("assistant") \
    .with_model("claude-3.5-sonnet") \
    .with_stack(["python", "analysis"]) \
    .build()
